# 智能提醒应用 Web版本

基于React + Vite的Web版本提醒应用,支持农历初一十五和周期性提醒。

## ✨ 特性

- 🌙 农历初一十五自动提醒
- 🔄 周期性提醒(每小时/每天/每周/每月)
- ⏰ 固定时间提醒
- ⭐ 自定义周期提醒
- 💾 本地存储持久化
- 🔔 浏览器通知支持
- 🎨 优雅的夜空主题设计
- 📱 响应式布局,支持移动端

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

应用会在 `http://localhost:3000` 打开。

### 构建生产版本

```bash
npm run build
```

构建后的文件在 `dist` 目录。

### 预览生产版本

```bash
npm run preview
```

## 🎯 功能说明

### 提醒类型

1. **农历提醒** 🌙
   - 农历初一或十五自动触发
   - 显示月相信息
   - 无需手动设置

2. **周期性提醒** 🔄
   - 每小时
   - 每天
   - 每周
   - 每月

3. **固定时间提醒** ⏰
   - 指定日期和时间
   - 一次性提醒

4. **自定义提醒** ⭐
   - 每周几(周一到周日)
   - 每月几号(1-31号)
   - 灵活配置

## 🎨 技术栈

- **框架**: React 18
- **构建工具**: Vite 5
- **样式**: CSS3 + CSS Variables
- **农历计算**: lunar-javascript
- **存储**: localStorage
- **通知**: Web Notifications API

## 💡 使用提示

### 启用浏览器通知

首次使用时,浏览器会请求通知权限,请点击"允许"以接收提醒通知。

### 数据持久化

所有提醒数据存储在浏览器的localStorage中,清除浏览器数据会丢失提醒。

### 与原生版本对比

Web版本功能与React Native版本基本一致,但有以下差异:
- 浏览器通知需要用户授权
- 不支持后台运行(需要浏览器保持打开)
- 不支持系统级通知
